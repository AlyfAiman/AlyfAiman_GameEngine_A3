On play, program will boot up with a Blue splash screen while console will detect that splash screen is
successfully activated.

After 5 seconds, splash screen will be deactivated while console detects it as well.

will then go to game scene (purple) while console detecting game scene has been activated.

at this time, 5 unique GUID numbers will also appear in the console.

while in game scene, user can pres "A" key to display the gameobject's name.

user can also use Left and right mouse button to display the coordinates clicked in the console.

when user presses "D" key, the game object containing the component will be deleted along with its components.
this will also be detected and shown in the console.
